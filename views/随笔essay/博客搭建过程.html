# 一、介绍

首先，我需要告诉你的是，其实搭建一个这样的博客网站，本身并不复杂，甚至可以说简单(当然，在你不自主开发主题的情况下)，你只需要了解 Js+Vue+VuePress 即可！

# 二、意义

本博客存在的意义:学习知识的记录，以及交流。

# 三、博客搭建过程

本博客是基于尤大大(尤雨溪)开发的 一款免费开源的 极简网站生成器 VuePress (opens new window) + vuepress-theme-reco (opens new window) 主题 构建而成，以及学习各优秀博客作品的产物。

# 四、VuePress 相关学习资料

  1. bilibili 一步步搭建 VuePress 及优化 (opens new window)
  2. bilibili 1 小时带你搭建 vuepress 高大上个人博客（极速上手版） (opens new window)

建议你两个视频都看。

# 五、VuePress-theme-reco 主题
